# tiku_rn

#### 介绍

有知学堂App，是一款专门为学生设计的证书备考手机应用。涵盖了会计类、计算机等级类、通信类等热门课程。

主要功能

-试题相关：包含每日一练、历年真题、模拟试卷、章节练习等功能模块，以及试题收藏、做题记录和错题库等辅助内容；

-资讯相关：包含考试资讯、报名时间、成绩查询等信息；

-个人相关：包括个人信息、账户信息、已购记录、收藏列表、设置等功能。

考证无忧，最贴心的考证助手！
